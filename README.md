# Haochong-week1-mini-repo
This is a repo template for course 706_Data_Engineering Week 1 Mini Project. I created all required documents first. Then, I add main.py and test.py by myself. I defined an get_sum function to see if I can get pass in the Action. It contains:

.devcontainer including a Dockerfile

workflows including GitHub Actions

vscode including settings

.gitignore

Makefile

README.md

main.py

requirements.txt

test_main.py

# Purpose
To make an template that can be used in the future projects. 

## Preparation 
1. open codespaces 
2. Built container and activate enviroment
3. run get_sum and check the status in Action
